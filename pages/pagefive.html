A rel attribute is use to describe the relationship between the current page and the linked.

The noopener value prevents the opened link from gaining access to the webpage from which is was opened. Basically, it's a security measure.

The noreferrer value prevents the opened link from knowing which webpage has a reference to it.

Newer versions of browsers provide this security. To err on the side of caution, always pair a target="_blank" with a rel="noopener noreferrer".
